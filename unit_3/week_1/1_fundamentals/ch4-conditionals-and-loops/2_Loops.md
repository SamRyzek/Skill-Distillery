### Loops
* `for`, `do...while` and `while` loop syntax is the same in JS as it is in Java

```javascript
var arr = [1,2,3,4];

// for loop
for (var i = 0 ; i < arr.length ; i++) {
	// do something
}

// while loop
while (false) {
	// do something
	// ... it won't
	// ... its test case evaluates to false
}

// do...while loop
do {
	// do something
	// ... it will do it once
} while(false);
```
